# 03_IPC_CONTRACT.md — IPC / Pipe Contract

> **Input:** INTERVIEW_OUTPUT.yaml (phân tích: nếu dự án có IPC)
> **Output:** Token format, timing, heartbeat, fallback behavior cho tất cả bên tham gia.
