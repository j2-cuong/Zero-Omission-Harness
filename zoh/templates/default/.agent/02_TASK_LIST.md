# 02_TASK_LIST.md — Danh sách task
> **Vai trò:** Lộ trình thực hiện chi tiết — từng task có input, output, dependency rõ ràng.
> **Input:** 01_STRUCTURE.md + INTERVIEW_OUTPUT.yaml (phân tích)
> **Output:** Task list có thứ tự, mỗi task có: file tạo ra, dependency, độ khó, agent phụ trách.
> **Rule:** Mỗi lần chỉ làm một task. Chờ confirm trước khi sang task tiếp. Update STATE.md sau mỗi task.

## Format mỗi task

```
| ID | Tên | File output | Depends on | Input | Output | Agent | Độ khó |
```
